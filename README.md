# SpendWise

SpendWise is a personal finance management Android application designed to help users track income, expenses, budgets, and savings goals in a clear and organized way. The app is built using Kotlin, Room Database, MVVM architecture, and Material Design 3. It provides analytical tools and a simple conversational assistant to help users understand and improve their financial behavior.

## Features

User authentication including login, registration, and password reset
Dashboard with visual analytics and monthly summaries
Transaction management with add, edit, and delete functions
Category management for income and expense types
Monthly budgets with category limits
Savings goals with progress tracking
Rule-based chatbot for financial guidance
Dark and light theme support
Local storage using Room Database

## Technologies Used

Kotlin
Android Studio
Room Database
MVVM Architecture
LiveData and ViewModel
Navigation Component
Material Design 3
MPAndroidChart

## Purpose

SpendWise helps users monitor their financial activities, control their spending, set financial goals, and build better financial habits. It is suitable for students, families, and individuals who want a simple and effective tool for managing their personal finances.
